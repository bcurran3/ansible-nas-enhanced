## Usage

Set `youtubedl-material_enabled: true` in your `inventories/<your_inventory>/nas.yml` file.

ane.sh --enable youtubedl-material to enable the app
ane.sh --up to install the app

If you want to access youtubedl-material externally, don't forget to set `youtubedl-material_available_externally: true` in your `inventories/<your_inventory>/nas.yml` file.
