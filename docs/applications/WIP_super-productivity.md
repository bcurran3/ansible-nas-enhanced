## Usage

Set `super-productivity_enabled: true` in your `inventories/<your_inventory>/nas.yml` file.

ane.sh --enable super-productivity to enable the app
ane.sh --up to install the app

If you want to access super-productivity externally, don't forget to set `super-productivity_available_externally: true` in your `inventories/<your_inventory>/nas.yml` file.
